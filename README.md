# How to deploy
```sh
# Clone repo
git clone https://github.com/tuyenldhust/ITSS2_Pomodoro.git

# Change to directory xingtu_gym
cd ITSS2_Pomodoro

chmod +x install-docker.sh script.sh

# Install docker and docker compose
./install-docker.sh

# Build
./script.sh build

# Start
./script.sh start

# Stop
./script.sh stop

# Clean
./script.sh clean

# Clean all
./script.sh fclean
```
