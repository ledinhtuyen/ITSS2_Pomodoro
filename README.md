# How to deploy
```sh
# Clone repo
git clone https://github.com/tuyenldhust/ITSS2_Pomodoro.git

# Change to directory xingtu_gym
cd ITSS2_Pomodoro

chmod +x install-docker.sh script.sh

# Install docker and docker compose
sudo ./install-docker.sh

# Build
sudo ./script.sh build

# Start
sudo ./script.sh start

# Stop
sudo ./script.sh stop

# Clean
sudo ./script.sh clean

# Clean all
sudo ./script.sh fclean
```
